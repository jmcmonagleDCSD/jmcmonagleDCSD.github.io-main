Computer Science

Websites

We will be using html and css to design websites from scratch.

code.org ‐ is a great resource if you missed days or for further practice at home.

Journal ‐ this is our updated journal if you've missed days or just need to update your journal.

w3schools ‐ is a great resource to learn new html tags, css properties for a list of properties, or click the html/css tab to learn about specific topics.

unsplash ‐ is good place to go if you want to find safe free to use images. You can also google images, but you must follow these steps:

  • click the image tab
  • click the tools button
  • click the usage rights drop dropdown
  • select creative common licenses
